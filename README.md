# Tetris watchface
<h3 align="center">Tetris 1989 watchface for Galaxy Watch 8</h3>

<img width="400" src="https://github.com/user-attachments/assets/f1506cbd-8c75-40b6-a135-d73e459a2a3a" alt="Material Bread logo">

В репозитории выложены все ассеты, которые используются в часах, а также сам файл .wfs для тех, кто захочит делать какие либо модификации.

<h3>Функции:</h3>
- AM\PM и 24Hr режимы<br> 
- Тап-зоны<br> 
- 4.5 минуты анимации<br> 
- Always-On режим<br> 
- Индикатор заряда, пульс, погода, цель по шагам, дата день месяц
